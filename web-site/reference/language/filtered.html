Filtered expressions

Filtered expressions are used to filter content of a collection.

Filtering a collection

Filtering a collection produces an Iterator value, which can then be used as an input for other constructs.
Such an expression has the form fetch any item from collection where predicate ( and/or predicate ), where item is a valid variable identifier, collection is a collection expression and predicate is a boolean expression in which item can be used to represent the current collection item being evaluated.

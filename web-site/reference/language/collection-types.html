Collection types

Collection types are technical types which can hold multiple values, typically of the same value type.

List

The List type is a technical type used to keep together 0 or more items of the same type.
The items themselves can be of any type.
A List type is declared by adding square brackets '[]' after the item type. 

Text[] -> the type Text list
A List literal must be enclosed in square brackets '[' and ']', the items being separated by a comma ','.
[12, 15, -3] -> a List of 3 Integers
["Paris", "London", "Singapore"] -> a List of 3 Texts

Items in a List can be accessed individually using their position.
Positions start at 1 and end at the list length.

a = [12, 15, -3] -> assign a List of 3 Integers to variable 'a'
a[1] -> get item at position 1 in List referenced by variable 'a' i.e. the Integer 12

Tuple

The Tuple type is a technical type used to keep together 0 or more items of different types.
The items themselves can be of any type.
A Tuple type is declared as follows. 

(,) -> the type Tuple
A Tuple literal must be enclosed in parenthesis '(' and ')', the items being separated by a comma ','.
For Tuple comprising 0 or 1 item, because the above could be interpreted as a parenthesized expression, a comma ',' must precede the right parenthesis.
(,) -> the empty Tuple
(3.14,) -> a Tuple of 1 item
("Paris", '2015-06-12', -113.87) -> a Tuple of 3 items

Items in a Tuple can be accessed individually using their position.
Positions start at 1 and end at the list length.

a = ("Paris", '2015-06-12', -113.87) -> assign a Tuple of 3 items to variable 'a'
a[1] -> get item at position 1 in Tuple referenced by variable 'a' i.e. the Text "Paris"

Set

The Set type is a technical type used to keep together 0 or more distinct items of the same type.
The items themselves can be of any type.
A Set type is declared by adding angle brackets '<>' after the item type. 

Text<> -> the type Text set
A Set literal must be enclosed in angle brackets '<' and '>', the items being separated by a comma ','.
<12, 15, -3> -> a set of 3 Integers
<"Paris", "London", "Singapore"> -> a Set of 3 Texts

Items in a Set can be accessed individually using their position.
However this is discouraged, because the position in the Set may change internally when ensuring uniqueness.
Set items are generally accessed using iterators or predicates.

s = <12, 15, -3> -> assign a Set of 3 Integers to variable 's'
for each i in s -> iterate over items in Set referenced by variable 's'
s contains 12 -> return true if Set referenced by variable 's' contains the Integer 12

Dictionary

The Dictionary type is a technical type used to keep together 0 or more items of the same type and access them using a Text key.
The items themselves can be of any type.
A Dictionary type is declared by adding curly brackets '{}' after the item type. 

Text{} -> the type Text dictionary
A Dictionary literal must be enclosed in curly brackets '{' and '}', the items being separated by a comma ',', the key and value being separated by a semi colon ':'.
{} -> an empty Dictionary
{ "Paris" : "France", "London" : "England", "Berlin" : "Germany" } -> a Dictionary of 3 keys and Text values

Items in a Dictionary can be accessed individually using their key.

d = { "Paris" : "France", "London" : "England" } -> assign a Dictionary of 2 Texts to variable 'd'
d["Paris"] -> get item with key "Paris" i.e. Dictionary referenced by variable 'd' i.e. "France"

Range

The Range type is a technical type used to create sequences of 1 or more items of the same discrete type.
The items themselves can only be of type Character, Integer, Date or Time.
A Range type is declared by adding 2 dots enclosed in square brackets '[..]' after the item type. 

Integer[..] -> the type Integer range
A Range literal must be enclosed in square brackets '[' and ']', the start and end value being separated by 2 dots '..'.
[1..17] -> all Integers ranging from 1 to 17 included
['c'..'k'] -> all Characters ranging from 'c' to 'k' included (using their Unicode value)
['2012-01-01'..'2014-04-01'] -> all Dates ranging from '2012-01-01' to '2014-04-01'
['20:01:01'..'23:02:02'] -> all Times ranging from '20:01:01' to '23:02:02', with a 1 second increment

Items in a Range can be accessed individually using their position.
Positions start at 1 and end at the Range length.

r = [2..17] -> assign a Range of Integers to variable 'r'
r[4] -> get item at position 4 in Range referenced by variable 'r' i.e. the Integer 5

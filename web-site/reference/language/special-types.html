Special types

Special types are technical types which do not hold information and only live during program execution.

Any

The Any type is a general technical type to represent any type of data.
This type is useful for describing value types when the actual type is not known at compile time.
It is also useful to hold opaque native values.

Method

The Method type is a technical type used to hold Prompto methods.
Method types are declared implicitly when declaring methods.
Prompto supports assigning methods to variables and passing them as parameters.

m = myMethod -> assign the method 'myMethod' to variable 'm'

Code

The Code type is a technical type used to represent Prompto code.
Prompto supports passing code snippets as parameters, so this is useful for passing code to methods.

c = Code: a + b -> assign the Prompto code 'a + b' to variable 'c'

Iterator

The Iterator type is a technical type produced by Prompto when transforming a collection inline.

x + 1 for each x in [1..5] -> returns an Iterator which will produce an item on each iteration

Cursor

The Cursor type is a technical type used to navigate through data returned by the data store.

c = fetch all -> returns a Cursor which will produce an item on each iteration

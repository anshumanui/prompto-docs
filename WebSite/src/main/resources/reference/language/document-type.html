Document

The Document type is a technical type used to manipulate arbitrary data structures with loose typing.
A Document has items which can be of any type, atomic, collection or Document.
A Document is useful to read data from the store, from files or from the Internet, and convert it to typed data.

Creating a Document

Creating a Document is done simply by using the Document keyword and assigning it to a variable.
Once the document is created, it can be populated simply by assigning values to members. Both member definition and member population occur at the same time i.e. a document has no predefined members.
